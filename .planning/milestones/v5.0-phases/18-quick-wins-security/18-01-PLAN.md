In `server/src/index.ts` die CORS-Registrierung (aktuell Zeilen 17-19) ersetzen:

```typescript
// Vorher:
await fastify.register(cors, {
  origin: process.env.CORS_ORIGIN ?? '*',
});

// Nachher:
const corsOrigin = process.env.CORS_ORIGIN;
if (!corsOrigin) {
  throw new Error('CORS_ORIGIN env var ist nicht gesetzt. Setze CORS_ORIGIN=https://fairstand.godsapp.de (oder mehrere Origins kommasepariert).');
}
const allowedOrigins = corsOrigin.split(',').map(o => o.trim());
await fastify.register(cors, {
  origin: (origin, callback) => {
    // Erlaubt Requests ohne Origin-Header (z.B. direkte Server-zu-Server-Calls, curl)
    if (!origin || allowedOrigins.includes(origin)) {
      callback(null, true);
    } else {
      callback(new Error(`Origin ${origin} ist nicht erlaubt`), false);
    }
  },
});
```

Kommaseparierte Werte erlauben mehrere Origins ohne Array-Syntax in der Docker-Umgebungsvariable (z.B. `CORS_ORIGIN=https://fairstand.godsapp.de,http://localhost:5173`).
