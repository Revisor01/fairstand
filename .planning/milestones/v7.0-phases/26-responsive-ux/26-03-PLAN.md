Füge Swipe-Gesten NUR für den Slide-In Modus (sidebar=false) hinzu. Der Sidebar-Modus (sidebar=true) bekommt keine Swipe-Geste.

**Konstante** (am Anfang der Datei, nach den Imports):
```tsx
const SWIPE_DISMISS_THRESHOLD = 60; // px — Mindestdistanz für Dismiss
```

**Neue State-Variablen** in CartPanel-Funktion (im Slide-In Zweig):
```tsx
const [swipeStartX, setSwipeStartX] = useState(0);
const [swipeOffset, setSwipeOffset] = useState(0);
const [isSwiping, setIsSwiping] = useState(false);
```

**Swipe-Handler** für das Slide-In Panel:
```tsx
function handlePanelPointerDown(e: React.PointerEvent<HTMLDivElement>) {
  if (!isOpen) return;
  setSwipeStartX(e.clientX);
  setIsSwiping(true);
  (e.currentTarget as HTMLDivElement).setPointerCapture(e.pointerId);
}

function handlePanelPointerMove(e: React.PointerEvent<HTMLDivElement>) {
  if (!isSwiping || !isOpen) return;
  const delta = e.clientX - swipeStartX;
  // Nur Swipe nach rechts (delta > 0) für Dismiss erlauben
  if (delta > 0) {
    setSwipeOffset(Math.min(delta, 400));
  }
}

function handlePanelPointerUp() {
  if (swipeOffset > SWIPE_DISMISS_THRESHOLD) {
    onClose();
  }
  setSwipeOffset(0);
  setIsSwiping(false);
}

function handlePanelPointerCancel() {
  setSwipeOffset(0);
  setIsSwiping(false);
}
```

**Panel-div im Slide-In Modus** — `style` und Handler ergänzen:
```tsx
<div
  className={`
    fixed inset-y-0 right-0 w-80 md:w-96 bg-white shadow-2xl z-50
    flex flex-col
    ${isSwiping ? '' : 'transform transition-transform duration-300'}
    ${isOpen ? 'translate-x-0' : 'translate-x-full'}
  `}
  style={isOpen && swipeOffset > 0
    ? { transform: `translateX(${swipeOffset}px)` }
    : undefined
  }
  onPointerDown={handlePanelPointerDown}
  onPointerMove={handlePanelPointerMove}
  onPointerUp={handlePanelPointerUp}
  onPointerCancel={handlePanelPointerCancel}
>
```

Wichtig: `transition-transform` nur wenn NICHT geswipt wird (`!isSwiping`), sonst ruckelt das Panel während der Geste.

**`onOpen` optionaler Prop** für Swipe-to-Open (wird in POSScreen gesetzt):
```tsx
interface CartPanelProps {
  // ... vorhandene Props ...
  onOpen?: () => void;  // Callback für Swipe-to-Open
}
```

**Swipe-to-Open** auf dem Overlay (wenn Panel geschlossen): Ein transparentes Edge-Element am rechten Bildschirmrand fängt Swipes ab. Füge AUSSERHALB des Panels (nach dem Fragment-Start) ein Edge-Element ein, das nur sichtbar ist wenn `!isOpen`:
```tsx
{!isOpen && onOpen && (
  <div
    className="fixed inset-y-0 right-0 w-8 z-30"
    onPointerDown={e => {
      const startX = e.clientX;
      const handleMove = (ev: PointerEvent) => {
        if (startX - ev.clientX > SWIPE_DISMISS_THRESHOLD) {
          onOpen();
          window.removeEventListener('pointermove', handleMove);
          window.removeEventListener('pointerup', handleClean);
        }
      };
      const handleClean = () => {
        window.removeEventListener('pointermove', handleMove);
        window.removeEventListener('pointerup', handleClean);
      };
      window.addEventListener('pointermove', handleMove);
      window.addEventListener('pointerup', handleClean);
    }}
    aria-hidden="true"
  />
)}
```

Hinweis: Das Edge-Element hat keine sichtbare Farbe (`bg-transparent`) und ist `aria-hidden`. Es ist nur ein Touch-Target am rechten Rand.
