## E2E Flow Verification

### Flow 1: User Logon (Online Required)
```
Client: PIN Dialog → serverLogin('/api/auth/pin')
  ↓
Server: auth.ts → await db.select().from(shops).where(eq(pin, pinHash))
  ↓
PostgreSQL: shops table lookup
  ↓
Server: createSession(token, shopId) → crypto.randomUUID()
  ↓
Client: localStorage.setItem('session', {shopId, token})
  ↓
Result: Ready for POS mode
```
**Status:** ✓ CONNECTED

### Flow 2: Product Display
```
Client: useProducts() → fetch('/api/products?shopId=X', headers)
  ↓
Server: products.ts → await db.select().from(products).where(shopId)
  ↓
PostgreSQL: products table filtered by shop
  ↓
Server: reply.send(rows) [snake_case from DB]
  ↓
Client: mapServerProduct() converts to camelCase
  ↓
TanStack Query: caches with key ['products', shopId]
  ↓
Result: ArticleGrid renders products
```
**Status:** ✓ CONNECTED

### Flow 3: Sale Completion
```
Client: PaymentFlow → completeSale(items, paidCents, changeCents)
  ↓
completeSale(): constructs Sale object, validates amounts
  ↓
Client: fetch('/api/sync', { entries: [{operation: SALE_COMPLETE, payload: sale}] })
  ↓
Server: sync.ts → Zod validation → shopId check
  ↓
Server: await db.transaction(async (tx) => {
  - await tx.insert(sales).values(...).onConflictDoNothing()
  - await tx.update(products).set({stock: stock - quantity})...
  - await tx.insert(outboxEvents)...
})
  ↓
Server: broadcast({type: 'stock_changed', shopId})
  ↓
Client: WebSocket listener → queryClient.invalidateQueries(['products', shopId])
  ↓
Client: useProducts() refetch → POSScreen re-renders with new stock
  ↓
Result: Sale recorded, stock updated on all connected devices
```
**Status:** ✓ CONNECTED

### Flow 4: Offline Detection
```
User: Disconnects internet
  ↓
Browser: emits 'offline' event
  ↓
POSScreen: window.addEventListener('offline', () => setIsOnline(false))
  ↓
Render: if (!isOnline) return OfflineOverlay
  ↓
UI: "Keine Internetverbindung" with instructions
  ↓
Result: User cannot proceed, clear error message
```
**Status:** ✓ CONNECTED

### Flow 5: Admin Daily Report
```
Client: DailyReport → useQuery({ queryFn: fetch('/api/sales?from=...&to=...') })
  ↓
Server: sales.ts → await db.execute(sql\`SELECT * FROM sales WHERE...\`)
  ↓
PostgreSQL: WHERE shop_id = X AND created_at BETWEEN from AND to
  ↓
Server: reply.send(result.rows)
  ↓
Client: mapServerSale() converts snake_case to Sale interface
  ↓
TanStack Query: caches with key ['sales', shopId, dayStart, dayEnd]
  ↓
DailyReport: renders sales table, stats, detail modal
  ↓
Result: Admin sees daily sales with correct amounts
```
**Status:** ✓ CONNECTED

### Flow 6: Live Product Update
```
Admin: Updates product price in ProductForm
  ↓
Client: useUpdateProduct().mutate({...product, price: 500})
  ↓
Server: products.ts → POST /api/products → onConflictDoUpdate (LWW)
  ↓
PostgreSQL: UPDATE products SET price=500, updatedAt=NOW() WHERE id=X
  ↓
Server: broadcast({type: 'products_changed', shopId})
  ↓
Client A (POS): WebSocket receives → queryClient.invalidateQueries(['products', shopId])
  ↓
Client A: useProducts() refetch → new price in ArticleGrid
  ↓
Result: All POS devices see new price within seconds
```
**Status:** ✓ CONNECTED
