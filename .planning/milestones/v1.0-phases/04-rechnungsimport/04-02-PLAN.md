**Upload-Handler (`handleFileSelected`):**
1. `setUploading(true)`, `setError(null)`, `setSuccessMessage(null)`
2. `FormData` erstellen, File anhaengen
3. `fetch('/api/import/parse', { method: 'POST', body: formData })` — KEIN Content-Type Header setzen (Browser setzt multipart boundary automatisch)
4. Response parsen: `{ rows: ParsedInvoiceRow[], filename: string }`
5. **Matching gegen Dexie:** Alle Produkte laden: `db.products.where('shopId').equals(SHOP_ID).toArray()`
6. Index aufbauen: `new Map(products.map(p => [p.articleNumber.toLowerCase().trim(), p]))`
7. Jede Row matchen: `status: match ? 'known' : 'new'`, `existingProductId: match?.id`, `checked: true`
8. `setRows(matchedRows)`, `setUploading(false)`
9. Bei Fetch-Fehler: `setError('Fehler beim Verarbeiten der PDF: ' + errorText)`, `setUploading(false)`

**Row-Change-Handler:**
```typescript
function handleRowChange(index: number, updates: Partial<MatchedRow>) {
  setRows(prev => prev.map((r, i) => i === index ? { ...r, ...updates } : r));
}
```

**Toggle-Handler:**
```typescript
function handleToggleCheck(index: number) {
  setRows(prev => prev.map((r, i) => i === index ? { ...r, checked: !r.checked } : r));
}
function handleToggleAll() {
  const allChecked = rows.every(r => r.checked);
  setRows(prev => prev.map(r => ({ ...r, checked: !allChecked })));
}
```

**Freigabe-Handler (`handleCommit`):**
1. `setCommitting(true)`, `setError(null)`
2. `const checkedRows = rows.filter(r => r.checked)`
3. Fuer jede Zeile:
   a. Wenn `status === 'new'`: Neues Produkt in Dexie anlegen:
      ```typescript
      const newProduct: Product = {
        id: crypto.randomUUID(),
        shopId: SHOP_ID,
        articleNumber: row.articleNumber.trim(),
        name: row.name.trim(),
        category: '',
        purchasePrice: row.purchasePriceCents,
        salePrice: row.evpCents ?? 0,
        vatRate: row.vatRate,
        stock: 0,
        minStock: 0,
        active: true,
        updatedAt: Date.now(),
      };
      await db.products.add(newProduct);
      ```
      Fire-and-forget Server-Sync (exakt wie ProductForm):
      ```typescript
      if (navigator.onLine) {
        fetch('/api/products', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify(newProduct),
        }).catch(() => {});
      }
      ```
   b. Bestandsbuchung via Outbox (STOCK_ADJUST mit positivem Delta):
      ```typescript
      await db.outbox.add({
        operation: 'STOCK_ADJUST',
        payload: {
          productId: row.status === 'known' ? row.existingProductId! : newProduct.id,
          delta: row.quantity,
          reason: 'Import Rechnung',
          shopId: SHOP_ID,
        },
        shopId: SHOP_ID,
        createdAt: Date.now(),
        attempts: 0,
      });
      ```
   c. Lokalen Bestand in Dexie direkt aktualisieren:
      ```typescript
      const productId = row.status === 'known' ? row.existingProductId! : newProduct.id;
      await db.products.where('id').equals(productId).modify(p => { p.stock += row.quantity; });
      ```

4. Outbox flushen: `if (navigator.onLine) { await flushOutbox(); }`
5. Import-Historie aktualisieren:
   ```typescript
   const history = (await get<ImportHistoryEntry[]>('import-history')) ?? [];
   history.unshift({ date: Date.now(), filename: currentFilename, positionCount: checkedRows.length });
   await set('import-history', history.slice(0, 50)); // max 50 Eintraege
   setImportHistory(history.slice(0, 50));
   ```
6. `setSuccessMessage(`${checkedRows.length} Positionen gebucht.`)`, `setRows([])`, `setCommitting(false)`

**Render:**
- Wenn `rows.length === 0` und kein Upload laeuft: UploadZone anzeigen
- Wenn `rows.length > 0`: ReviewTable anzeigen + "Bestand buchen"-Button + "Abbrechen"-Button
- "Bestand buchen"-Button: `disabled={committing || !rows.some(r => r.checked)}`, Tailwind: `bg-sky-500 hover:bg-sky-600 active:bg-sky-700 disabled:opacity-50 text-white font-semibold h-14 text-lg rounded-xl w-full`
- "Abbrechen"-Button: `text-slate-500 hover:text-slate-700`, setzt `rows` auf `[]`
- Erfolgs-Meldung: `bg-green-50 text-green-700 px-4 py-3 rounded-xl`
- Fehler-Meldung: `bg-rose-50 text-rose-700 px-4 py-3 rounded-xl` (wie ProductForm)
- Import-Historie unter Upload-Zone: Einfache Liste (Datum formatiert, Dateiname, Anzahl Positionen)
