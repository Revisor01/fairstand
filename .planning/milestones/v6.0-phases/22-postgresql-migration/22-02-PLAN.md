**server/src/routes/sync.ts** — Transaktionen auf async umstellen:

Für alle 4 Operationen (SALE_COMPLETE, STOCK_ADJUST, SALE_CANCEL, ITEM_RETURN) jeweils:

VORHER Muster:
```typescript
db.transaction((tx) => {
  tx.insert(sales).values({ ... }).run();
  for (const item of sale.items) {
    tx.update(products).set({ ... }).where(...).run();
  }
  tx.insert(outboxEvents).values({ ... }).run();

```

NACHHER Muster — `await` davor, `async (tx)`, alle `.run()` entfernen, `await` vor jeden TX-Call:
```typescript
await db.transaction(async (tx) => {
  await tx.insert(sales).values({ ... });
  for (const item of sale.items) {
    await tx.update(products).set({ ... }).where(...);
  }
  await tx.insert(outboxEvents).values({ ... });
});
```

Vollständiges Beispiel für SALE_COMPLETE Block nach Änderung:
```typescript
await db.transaction(async (tx) => {
  await tx.insert(sales).values({
    id: sale.id,
    shopId: sale.shopId,
    items: sale.items,
    totalCents: sale.totalCents,
    paidCents: sale.paidCents,
    changeCents: sale.changeCents,
    donationCents: sale.donationCents,
    type: sale.type ?? null,
    createdAt: sale.createdAt,
    syncedAt: Date.now(),

  for (const item of sale.items) {
    await tx.update(products)
      .set({ stock: sql`${products.stock} - ${item.quantity}` })
      .where(eq(products.id, item.productId));
  }

  await tx.insert(outboxEvents).values({
    shopId: entry.shopId,
    operation: entry.operation,
    payload: entry.payload,
    processedAt: Date.now(),
    createdAt: entry.createdAt,
  });
});
```

**server/src/routes/reports.ts** — SQLite-Raw-SQL durch PostgreSQL ersetzen:

Mapping-Tabelle (jede SQLite-Funktion hat genau ein PostgreSQL-Äquivalent):
- `json_extract(item.value, '$.fieldName')` → `item->>'fieldName'` (gibt text zurück)
- `CAST(json_extract(item.value, '$.quantity') AS INTEGER)` → `(item->>'quantity')::int`
- `json_each(sales.items) as item` → `jsonb_array_elements(sales.items) as item`
- `strftime('%m', datetime(created_at / 1000, 'unixepoch'))` → `EXTRACT(MONTH FROM to_timestamp(created_at / 1000.0))::int`
- `db.all(sql\`...\`)` → `(await db.execute(sql\`...\`)).rows`

Konkrete Queries für reports.ts:

```sql
       COALESCE(SUM(total_cents), 0) as total_cents,
```

```sql

WHERE sales.shop_id = ${shopId}
  AND sales.created_at >= ${monthStart}
  AND sales.created_at < ${monthEnd}
  AND sales.cancelled_at IS NULL
  AND (sales.type IS NULL OR sales.type = 'sale')
```

**GET /reports/monthly topArticles Query:**
```sql
SELECT item->>'name' as name,
       SUM((item->>'quantity')::int) as total_qty,

WHERE sales.shop_id = ${shopId}
  AND sales.created_at >= ${monthStart}
  AND sales.created_at < ${monthEnd}
  AND sales.cancelled_at IS NULL
```
