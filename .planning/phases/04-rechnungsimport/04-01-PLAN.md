   **Exports:**
   - `ParsedInvoiceRow` Interface:
     ```typescript
     export interface ParsedInvoiceRow {
       lineNumber: number;
       quantity: number;
       articleNumber: string;
       name: string;
       purchasePriceCents: number;
       evpCents: number | null;
       vatRate: 7 | 19;
       parseWarning?: string;
     }
     ```
   - `parseSuedNordKontorPdf(buffer: Buffer): Promise<ParsedInvoiceRow[]>`

   **Import:** `import * as pdfjsLib from 'pdfjs-dist/legacy/build/pdf.mjs';`
   Dieser Import-Pfad ist PFLICHT fuer Node.js — der Standard-Import schlaegt fehl (Pitfall 1 aus RESEARCH.md).

   **Internes PdfTextItem Interface:**
   ```typescript
   interface PdfTextItem {
     str: string;
     transform: number[]; // [scaleX, skewX, skewY, scaleY, x, y]
     width: number;
   }
   ```

   **groupByRows(items, tolerance=2):** Text-Items nach Y-Koordinate (`transform[5]`) gruppieren. Y absteigend sortieren (PDF: oben=gross). Innerhalb jeder Zeile nach X-Koordinate (`transform[4]`) aufsteigend sortieren.

   **isInvoiceRow(row):** Erste Text-Item der Zeile matched `/^\d+\.$/` (laufende Nummer mit Punkt).

   **isHeaderRow(row):** Zeile enthaelt "Artikelnummer" oder "Bezeichnung" — Tabellenkopf ueberspringen.

   **parseEuroCents(str):** Euro-String zu Cent-Integer:
   - `str.replace('€', '').replace(/\s/g, '').replace(',', '.')` dann `Math.round(parseFloat(…) * 100)`
   - Return `null` bei NaN oder negativen Werten

   **parseMwSt(str):** Extrahiert 7 oder 19 aus "N %" Pattern. Return `null` bei anderem Wert.

   **extractEvp(designation):** EVP aus Bezeichnung extrahieren:
   - Regex: `/EVP\s*€\s*([\d]+[,.][\d]{2})/i`
   - Klammer-Block mit EVP aus Name entfernen: `.replace(/\s*\([^)]*EVP[^)]*\)/i, '')`
   - Return `{ cleanName: string, evpCents: number | null }`

   **Mehrzeilige Bezeichnungen (Pitfall 2):**
   Nach einer erkannten Rechnungszeile (laufende Nummer): Folgezeilen OHNE laufende Nummer, deren Items im X-Bereich der Bezeichnungsspalte liegen, als Fortsetzung des `name`-Felds behandeln. Bis zu 2 Fortsetzungszeilen sammeln. EVP-Extraktion erst NACH dem Zusammenfuegen aller Bezeichnungszeilen.

   **parseSuedNordKontorPdf:** Alle Seiten iterieren (`pdf.numPages`), pro Seite `getTextContent()` aufrufen, Zeilen gruppieren, Header ueberspringen, Rechnungszeilen parsen. Spalten-Zuordnung via X-Koordinaten-Reihenfolge der Items in einer Zeile (nicht fixe Pixel-Werte):
   - Item 0: laufende Nummer (wird verworfen)
   - Item 1: Menge (`parseInt`)
   - Item 2: Artikelnummer
   - Items 3..N-3: Bezeichnung (zusammengefuegt mit Leerzeichen)
   - Item N-2 oder vorletztes Euro-Item: Preis/St. (EK nach Rabatt)
   - Item N-1 oder letztes Prozent-Item: MwSt
   - Letztes Euro-Item: Gesamt (wird nicht gespeichert)

   Rabatt-Spalte kann leer sein oder "30 %" enthalten — wird uebersprungen. Entscheidend: `Preis/St.` ist der EK-Preis nach Rabatt.

   Bei Parse-Fehlern pro Zeile: `parseWarning` setzen statt Exception werfen. Zeile trotzdem zurueckgeben mit dem was geparst werden konnte.

1. Neue Datei `server/src/routes/import.ts` erstellen:

   ```typescript
   import type { FastifyInstance } from 'fastify';
   import multipart from '@fastify/multipart';
   import { parseSuedNordKontorPdf } from '../lib/pdfParser.js';

   export async function importRoutes(fastify: FastifyInstance) {
     await fastify.register(multipart, {
       limits: {
         fileSize: 10 * 1024 * 1024, // 10 MB max
         files: 1,
       },
     });

     fastify.post('/import/parse', async (request, reply) => {
       const data = await request.file();
       if (!data) {
         return reply.status(400).send({ error: 'Keine Datei uebermittelt' });
       }
       if (!data.filename.toLowerCase().endsWith('.pdf')) {
         return reply.status(400).send({ error: 'Nur PDF-Dateien erlaubt' });
       }

       try {
         const buffer = await data.toBuffer();
         const rows = await parseSuedNordKontorPdf(buffer);
         return reply.send({ rows, filename: data.filename });
       } catch (err) {
         fastify.log.error(err, 'PDF-Parsing fehlgeschlagen');
         return reply.status(422).send({
           error: 'PDF konnte nicht verarbeitet werden',
           detail: err instanceof Error ? err.message : String(err),
         });
       }
     });
   }
   ```
