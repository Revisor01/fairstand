**server/src/routes/reports.ts** — SQLite-Raw-SQL durch PostgreSQL ersetzen:

PostgreSQL-Äquivalente für SQLite-Funktionen:
- `json_extract(item.value, '$.fieldName')` → `item->>'fieldName'` (gibt text zurück)
- `CAST(json_extract(item.value, '$.quantity') AS INTEGER)` → `(item->>'quantity')::int`
- `json_each(sales.items) as item` → `jsonb_array_elements(sales.items) as item`
- `strftime('%m', datetime(created_at / 1000, 'unixepoch'))` → `EXTRACT(MONTH FROM to_timestamp(created_at / 1000.0))::int`
- `db.all(sql\`...\`)` → `await db.execute(sql\`...\`)` (Drizzle node-postgres: execute gibt {rows} zurück) ODER direkter: `(await db.execute(sql\`...\`)).rows`

Konkrete Queries für reports.ts:

WHERE sales.shop_id = ${shopId}

  AND (sales.type IS NULL OR sales.type = 'sale')

SELECT item->>'name' as name,
