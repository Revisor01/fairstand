**server/src/routes/categories.ts:** Neue Datei mit `categoryRoutes(fastify)`. Importiert `db` aus `../db/index.js`, `categories` und `products` aus `../db/schema.js`, `eq`, `sql`, `and` aus `drizzle-orm`.

Routen:
- `GET /categories?shopId=xxx` — alle Kategorien des Shops, sortiert nach `sortOrder`, dann `name`. Gibt Array zurück (camelCase: id, shopId, name, sortOrder, createdAt).
- `POST /categories` — Body: `{ shopId, name }`. Validierung: name nicht leer (400 bei Fehler). Erstellt mit `id = crypto.randomUUID()`, `createdAt = Date.now()`, `sortOrder = 0`. Gibt `{ ok: true, id }` zurück.
- `PATCH /categories/:id` — Body: `{ name }`. Lädt Kategorie, holt `oldName`. Führt Bulk-Update durch: `UPDATE products SET category = :newName WHERE shop_id = :shopId AND category = :oldName`. Dann `UPDATE categories SET name = :newName WHERE id = :id`. Gibt `{ ok: true }` zurück.
- `DELETE /categories/:id` — Lädt Kategorie. Prüft per COUNT-Query wie viele Produkte `category = cat.name AND shop_id = cat.shopId` haben. Wenn > 0: 409 mit `{ error: 'Kategorie wird von N Produkt(en) verwendet. Bitte zuerst Produkte umkategorisieren.' }`. Sonst löschen und `{ ok: true }` zurückgeben.

Alle Routen: 404 wenn Kategorie nicht existiert.

```typescript
export async function downloadCategories(): Promise<void> {
  const shopId = getShopId();
  const res = await fetch(`/api/categories?shopId=${shopId}`);
  if (!res.ok) throw new Error(`Kategorie-Download fehlgeschlagen: ${res.status}`);
  const serverCats = await res.json() as Category[];
  await db.transaction('rw', db.categories, async () => {
    await db.categories.where('shopId').equals(shopId).delete();
    await db.categories.bulkPut(serverCats);
  });
}
```
